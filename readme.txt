Replaces default coupon expiry field with dedicated tab for coupon validity settings.

== Description ==
This plugin replaces default coupon expiry field with dedicated tab where fields for setting coupon validity start date and time along with expiry date and time are shown.

New "Valid from" column is added to coupons list in admin panel and "Expiry date" column is tweaked to show expiry time.

== Installation ==
1. Download the plugin.
2. Unzip the woocommerce-coupon-schedule compressed file.
3. Upload the directory woocommerce-coupon-schedule to the "/wp-content/plugins" directory on your WordPress blog.
4. Activate the plugin through the "Plugins" menu in WordPress.
5. Visit WooCommerce Coupons section to see the plugin in action.
